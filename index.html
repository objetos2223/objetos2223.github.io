Contenido de la página del proyecto

La página de cada proyecto al menos debe contener los siguientes elementos:

  • objetivo general del proyecto
  • explicación de las reglas del juego, niveles, bonificaciones, obstáculos, etc.
  • enlace para descargar el archivo ejecutable
  • enlace al diagrama de clases UML
  • enlace al video en YouTube
  • enlace al proyecto en publicado en la página de Greenfoot
  • créditos (autores, fecha, materia, etc)

Puedes consultar el siguiente tutorial de páginas GitHub para conocer como modificar tu página.
